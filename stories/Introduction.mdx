# EchoMail Component Library

Welcome to the EchoMail component documentation. This Storybook contains all the UI components used in the EchoMail email marketing platform.

## Getting Started

Browse the sidebar to explore our components. Each component includes:

- **Interactive examples** - Play with different props and states
- **Documentation** - Learn how to use each component
- **Source code** - View the implementation details

## Component Categories

### UI Components

Core building blocks like buttons, inputs, cards, and dialogs.

### Email Components

Specialized components for email composition and preview.

### Layout Components

Navigation, headers, and page structure components.

## Design Principles

1. **Accessibility First** - All components are built with WCAG 2.1 AA compliance in mind
2. **Dark Mode Support** - Every component works seamlessly in both light and dark themes
3. **Responsive Design** - Components adapt to all screen sizes
4. **Type Safety** - Full TypeScript support with comprehensive prop types
